Crime Analyzer

Social media and local news websites were crawled to retrieve articles about criminal happenings in Ernakulam, Kerala. Analysis was done over the articles to aid the police department. Twitter API, Facebook graph API and custom crawlers were used in data retrieval process. A classifier was built using NLTK with categorized training data to segregate crime related articles. The entire process is planned to automate and update itself on a regular basis. Interaction with the application was done with a web interface, making it accessible across different locations.

Health Care (Predictive analysis solution)

The developed model gives the efficiency of a hospital with insights of areas where it can improve. Insurance companies were the players who will be benefited out of this analysis. Historical patient admission records were analyzed using elasticsearch and kibana. The restructured data was refined using Principle component analysis (PCA) and further analysis was done for getting level of contributions towards the list of features. A web interface was developed using chartjs for visualizing the end result.

Scheme search
